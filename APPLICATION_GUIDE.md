# Product Trial Application Guide

## How to Apply

We are currently accepting applications for the **Enterprise Multi-Agent System Technology Starter Kit V1.0** trial (April 2-10, 2026).

### Option 1: GitHub Issue (Recommended)
1. Go to the [Issues section](https://github.com/718haxi/718haxi.github.io/issues) of this repository
2. Click **"New Issue"**
3. Use the following title: **"Trial Application: [Your Name/Username]"**
4. Copy and paste the application template below into the issue body
5. Fill out all sections with your information
6. Click **"Submit new issue"**

### Option 2: Email Application
If you prefer not to use GitHub Issues, you can send your application to:
- **Email**: trial-application@selfevolving.cn (monitored daily)

Please use the same application template and send it as plain text in the email body.

## Selection Process

1. **Application Review**: We review all applications within 24 hours
2. **Selection Criteria**: Based on technical match, clear learning goals, and availability
3. **Notification**: Selected participants will receive:
   - GitHub repository access instructions
   - Discord community invitation
   - Welcome package and learning schedule
4. **Trial Start**: April 2, 2026
